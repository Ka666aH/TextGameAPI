# TextGame - Текстовая Ролевая Игра

Текстовая ролевая игра с REST API, разработанная на **ASP.NET Core** (.NET 8).

## Описание

Это серверная часть текстовой ролевой игры, где игроки могут:
- Исследовать комнаты и локации
- Сражаться с врагами
- Собирать предметы и управлять инвентарём
- Использовать ключи и монеты
- Взаимодействовать с объектами игрового мира

## Технологии

- **.NET 8** - платформа разработки
- **ASP.NET Core** - веб-фреймворк
- **Swagger/OpenAPI** - документация API
- **Controller-based архитектура** - RESTful API

### Доступ к API

После запуска приложение будет доступно по адресу:
- **HTTP**: `http://localhost:5000`
- **HTTPS**: `http://localhost:5001`

**Swagger UI** (документация API): `http://localhost:5000/swagger`
